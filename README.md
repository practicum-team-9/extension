# Описание проекта 
Это расширение для браузера, которое упрощает работу с ЯФормами для слабовидящих пользователей. 
## Функциональные возможности
### 1. Оставляет только один вопрос на странице
Скрывает вопросы и оставляет толькодин вопрос, чтобы упростить работу Screen Reader (Читалкам)

### 2. Добавляет кнопки по навигации по вопросам
Добавляет кнопки для взаимодействия с вопросами 


### 3. Добавляет возможности для взаимодействия с клавиатурой
Автоматически выбирает поле ввода. Нажатие на **ENTER** подтверждает заполнение. Нажатие на **ПРОБЕЛ** позволяет нажать на выбранный элемент (Кнопку или выпадаемый список)

### 4. Добавляет озвучку 
При отсутствии у пользователя читалки (программы - Screen Reader-а) добавляет озвучку вопроса посредством WebSpeech

# Установка
## Google Chrome / Chromium 
### Архив
 - Скачать архив с приложением и распаковать его.
 - В браузере перейти по ссылке (Яндекс браузер) browser://extensions/ (Google Chrome) chrome://extensions/ включить режим разработчика
 - Нажать "Load Unpacked" и выбрать папку с распакованным архивом

### Файл
 - Скачать файл расширения
 - В браузере перейти по ссылке (Яндекс браузер) browser://extensions/ (Google Chrome) chrome://extensions/ включить режим разработчика
 - Перенести файл в расширения

## Яндекс Браузер
### Архив
 - Скачать архив с приложением и распаковать его.
 - В браузере перейти по ссылке browser://extensions/ включить режим разработчика. ВНИМАНИЕ! Не отключать режим разработчика т.к. расширение работать не будет.
 - Нажать "Load Unpacked" и выбрать папку с распакованным архивом

### Файл
 - Скачать файл расширения
 - В браузере перейти по ссылке browser://extensions/ включить режим разработчика. ВНИМАНИЕ! Не отключать режим разработчика т.к. расширение работать не будет.
 - Перенести файл в расширения

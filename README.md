# YaAccessibility Extension


# Описание проекта 
## Возможности
Это расширение для браузера, которое упрощает работу с ЯФормами для слабовидящих пользователей. 
## Предполагаемый Функционал
 - Оставляет только один вопрос на странице, что позволяет проще найти
 - Добавляет кнопки для навигации по вопросам
 - Добавляет 
 - Добавляет озвучку


# Установка
## Скачать единый файл
 - Скачать файл.
 - В браузере включить режим разработчика
 - Перенести файл в браузер

## Скачать репозиторий с GitHub 

- Склонировать репозиторий
- В бразуере перейти на chrome://extensions/
- Включить Режим разработки и нажать Загрузить распокованное расширение.
- Выбрать папку с расширением и обавить его.
